Retrospective Questions

  1. How many hours did it take you to complete this assessment? (Please keep try to keep track of how many hours you have spent working on each individual part of this assessment as best you can - an estimation is fine; we just want a rough idea.)
    I did not keep exact track of the time I spent on this assignment. I always had it open and would do it here and there throughout the day. There were times when I sat down and focused on it for longer periods of time as well. I did some extra tasks to keep me motivated. I am more interested in back-end than front-end, but this semester has definitely given some inspiration for front-end. If I had just stuck with HTML and CSS and done the minimum, it would have probably taken 6 hours. I would say I spent about 10 hours on this QAP including research and gathering media (photos, video, information, learning some JavaScript elements, etc.)
  2. What online resources you have used? (My lectures, YouTube, Stack overflow etc.)
    I have used an abundance of information from the class lectures (Programming Concepts with JavaScript, Front-end, UI/UX) as well as some events I attended recently with Canada Learning Code and Get Coding.
    Here is a list of websites I used to obtain knowledge for some things I did:
    • Geeks for Geeks (style alert box)
    • W3 Schools (css tooltip)
    • Sololearn (scroll overflow)
    • Free Code Camp (css background image)
    • Flexbox Froggy
    • O'Reilly (Modern JavaScript From The Beginning)
  3. Did you need to ask any of your friends in solving the problems. (If yes, please mention name of the friend. They must be amongst your class fellows.)
    I didn't ask my friends for help directly, but I did ask many classmates for gentle stylistic feedback.
  4. Did you need to ask questions to any of your instructors? If so, how many questions did you ask (or how many help sessions did you require)?
    I did not ask my instructors for help. I did however use AI tools throughout the assignment. GitHub Copilot helped with some small repetitive tasks and I did ask ChatGPT several questions (specifically about embedding a YouTube video, getting my ToolTip to work properly, getting an opaque background on this page, and teaching me how to do the calculator in the pricing section).

    I would definitely consider this level-appropriate for QAP 1 of semester 2. I didn't have any moments where I felt frustrated or wanted to give up. The most difficult part for me was getting myself interested in cars, but once I found the right car the inspitation came quickly. After completing this task, I am confident that I could create a similar-quality website in a fraction of the time.
